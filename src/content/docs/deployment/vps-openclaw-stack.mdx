# VPS OpenClaw Stack

A self-hosted stack changes the value proposition of an automation system.

## Why VPS hosting matters

Running on a VPS means the system can stay on, keep context warm, and remain reachable without depending on a local laptop being awake. It also makes the project feel more like real infrastructure than a toy demo.

## Stack shape

A practical shape looks like this:

- VPS host
- OpenClaw gateway
- isolated agents and subagents
- repo workspaces and memory/state
- optional public-facing site or operator surfaces

## Operating principles

- keep public-facing surfaces separate from privileged internals
- avoid exposing dashboards casually
- use HTTPS, auth, and controlled routing
- treat uptime and safety as part of product quality

## What this signals

A self-hosted stack signals operational maturity: privacy awareness, infrastructure ownership, and a willingness to think beyond just prompt output quality.
